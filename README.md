# Mitropolia Moldovei și Bucovinei - Website

## 📖 Descripție
Acest proiect este site-ul oficial al Mitropoliei Moldovei și Bucovinei, o instituție religioasă ortodoxă din România.

## 📁 Structura Proiectului
```
.
├── index.html           # Pagina principală
├── css/
│   └── style.css       # Stiluri CSS
├── js/
│   └── script.js       # Funcționalități JavaScript
└── README.md           # Documentație
```

## 🎨 Caracteristici
- ✅ Design modern și responsiv
- ✅ Navigație intuitivă
- ✅ Carousel automat cu texte
- ✅ Social media links (YouTube, Facebook)
- ✅ Butoane de contact
- ✅ Optimizat pentru mobile

## 🚀 Cum să folosești

### 1. Clone Repository
```bash
git clone https://github.com/bahnareanudarius0-cloud/Mitropolia-Moldovei-si-Bucovinei.git
cd Mitropolia-Moldovei-si-Bucovinei
```

### 2. Deschide în Browser
- Deschide `index.html` direct în browser, sau
- Folosește [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) în VS Code

### 3. Personalizare
Editează aceste secțiuni în `index.html`:

#### Adaugă logo-ul tău
```html
<img src="https://via.placeholder.com/100x100?text=Logo" alt="Church Logo">
```
Înlocuiește URL-ul cu calea la logo-ul tău.

#### Adaugă imagine de fundal
```html
<img src="https://via.placeholder.com/1920x1080?text=Church+Background" alt="Background">
```
Înlocuiește URL-ul cu calea la imaginea ta.

#### Adaugă link-urile sociale
```html
<a href="https://youtube.com/@YourChannel" target="_blank"><i class="fab fa-youtube"></i></a>
<a href="https://facebook.com/YourPage" target="_blank"><i class="fab fa-facebook"></i></a>
```

#### Modifică texte
Editează textul în fiecare secțiune după cum ai nevoie.

1. Mergi la Settings al repository-ului
2. Selectează **Pages** din sidebar
3. Sub "Build and deployment" selectează **main** branch
4. Site-ul tău va fi live la: `https://bahnareanudarius0-cloud.github.io/Mitropolia-Moldovei-si-Bucovinei/`
